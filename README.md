# cyart-df
Digital Forensics Tasks (Week 3)
## Overview

This repository contains practical and theoretical work related to Digital Forensics including investigation process, malware analysis, and evidence handling.
## Tasks Covered

* Task 1: Chain of Custody
* Task 2: Investigation Process
* Task 3: Evidence Handling
* Task 4: Documentation Methods
* Task 5: Legal Considerations
* Task 6: Tools Introduction
* Task 7: Static Malware Analysis

## 🛠 Tools Used

* Autopsy
* FTK Imager
* HashMyFiles
* Wireshark
* VirusTotal

## Objective

To understand and implement digital forensic techniques including evidence handling, malware analysis, and investigation workflows.
